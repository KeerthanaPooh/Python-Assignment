## 1. What is Python and why is it called an interpreted language?
Python is a high-level, general-purpose programming language. It is called an interpreted language because the code is executed line by line by an interpreter instead of being compiled into machine code beforehand.

## 2. What are the key features of Python that make it popular?
- Easy to learn and use
- Readable syntax
- Interpreted language
- Object-oriented
- Large standard library
- Cross-platform support

## 3. Difference between Python 2 and Python 3
- Python 2 is outdated, Python 3 is the latest version
- Print statement vs print() function
- Better Unicode support in Python 3
- Improved libraries and features in Python 3

## 4. Applications of Python
- Web development
- Data science and AI/ML
- Automation and scripting
- Game development
- Desktop applications

## 5. What is PEP 8?
PEP 8 is the style guide for writing Python code. It ensures code readability and consistency.

## 6. Who developed Python and when?
Python was developed by Guido van Rossum and released in 1991.

## 7. What is dynamically typed?
In Python, variable types are decided at runtime. You don’t need to declare the type explicitly.

## 8. Compiler vs Interpreter
A compiler converts the whole program into machine code before execution. An interpreter executes code line by line. Python uses an interpreter.
